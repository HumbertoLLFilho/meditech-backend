## 🛠️ Tecnologias

- **Python 3.12** + **Flask**
- **SQLite**
- **Docker Compose**

## 🚀 Como Executar

### Com Docker Compose (recomendado)
`bash
docker-compose up --build
`

### Localmente
`bash
pip install -r requirements.txt
python run.py
`

Acesse: http://localhost:5000
